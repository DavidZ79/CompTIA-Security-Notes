A security control is a safeguard or countermeasure designed to protect the confidentiality, integrity, and availability (CIA triad) of information systems and data.


Control Categories

Technical Controls: Implemented through technology / technical systems of the organization
Ex: Firewalls, Data Encryption

Managerial Controls: Implemented by management through policies, procedures, and practices to direct individuals and teams.
Ex: Performance reviews, Risk assessments, Code of conduct

Operational Controls: Implemented with day-to-day processes
Ex: Incident response procedures, Security awareness training, User access management

Physical Controls: Implemented using tangible measures to protect organization assets
Ex: Access control vestibule
Biometric locks
Guards / security personnel
Security fences
CCTV surveillance systems
Mantraps
Vehicle barriers
Tamper-evident seals
Panic buttons/alarms


Control Types

Preventative controls

Deterrent controls

Detective controls

Corrective controls: Implemented after an incident to mitigat the impact

Compensating controls: Alternative measures to compensate for inadequate primary controls

Directive controls: Policies and regulations
