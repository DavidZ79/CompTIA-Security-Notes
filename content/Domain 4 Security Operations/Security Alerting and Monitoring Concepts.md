## Monitoring Computing Resources

Systematic approach to monitoring and securing organizational assets:

<details>
<summary><strong>Monitoring Components</strong></summary>

*Essential elements of comprehensive security monitoring*

**Monitoring Elements:**
- **Log Files** - System and application event logs
- **Security Logs** - Security-specific event logging
- **System Monitors** - Operating system monitoring
- **Application Monitors** - Application performance and security monitoring
- **Infrastructure Monitors** - Network and infrastructure monitoring

</details>

<details>
<summary><strong>Network Management</strong></summary>

*SNMP-based network monitoring and management*

**SNMP Components:**
- **SNMP Agents** - Devices that collect and report data
- **SNMP Managers** - Central management systems
- **SNMP Traps** - Asynchronous notifications from agents
- **Network Management System** - Centralized network monitoring platform

</details>

## Intrusion Detection and Prevention

Advanced threat detection and prevention systems:

<details>
<summary><strong>Network Intrusion Detection System (NIDS)</strong></summary>

*Passive network monitoring and threat detection*

**NIDS Capabilities:**
- **Traffic Analysis** - Analyzing network traffic patterns
- **Alerts and Notifications** - Generating security alerts
- **Passive Role** - Monitoring without blocking traffic

</details>

<details>
<summary><strong>Network Intrusion Prevention System (NIPS)</strong></summary>

*Active network security and threat prevention*

**NIPS Capabilities:**
- **Real-time Analysis** - Immediate threat detection and response
- **Threat Blocking** - Actively preventing malicious traffic
- **Policy Enforcement** - Enforcing security policies
- **Attack Surface Reduction** - Minimizing exposure to threats
- **Alerts and Reporting** - Comprehensive security reporting

</details>

## Security Operations Activities

Core activities in security monitoring and response:

<details>
<summary><strong>Monitoring Activities</strong></summary>

*Essential security operations tasks*

**Key Activities:**
- **Log Aggregation** - Centralizing log data from multiple sources
- **Alerting** - Generating and managing security alerts
- **Scanning** - Regular security assessments
- **Reporting** - Creating security reports and summaries
- **Compliance Reporting** - Meeting regulatory requirements
- **Executive Summaries** - High-level security briefings
- **Archiving** - Long-term storage of security data

</details>

<details>
<summary><strong>Alert Response and Validation</strong></summary>

*Managing and responding to security alerts*

**Response Activities:**
- **Quarantine** - Isolating compromised systems
- **Alert Tuning** - Optimizing alert sensitivity and accuracy

</details>

## Security Monitoring Tools

Essential tools for comprehensive security monitoring:

<details>
<summary><strong>Core Security Tools</strong></summary>

*Primary tools for security monitoring and management*

**Security Tools:**
- **SIEM** - Security Information and Event Management
- **SCAP** - Security Content Automation Protocol
- **Vulnerability Scanners** - Automated vulnerability assessment
- **Antivirus** - Malware detection and prevention
- **DLP** - Data Loss Prevention systems

</details>

<details>
<summary><strong>Network Monitoring Tools</strong></summary>

*Tools for network traffic and performance monitoring*

**Network Tools:**
- **SNMP Traps** - Network device notifications
- **NetFlow** - Network traffic flow analysis
- **Agents/Agentless** - Monitoring deployment methods
- **Benchmarks** - Security configuration standards

</details>

> [!note] Vocab Bank
> - **NIDS** - Network Intrusion Detection System, passive threat monitoring
> - **NIPS** - Network Intrusion Prevention System, active threat prevention
> - **SNMP** - Simple Network Management Protocol, network monitoring standard
> - **SIEM** - Security Information and Event Management, centralized security monitoring
> - **SCAP** - Security Content Automation Protocol, security configuration framework
> - **DLP** - Data Loss Prevention, systems to prevent data exfiltration
> - **NetFlow** - Network traffic flow monitoring and analysis
> - **SNMP Trap** - Asynchronous notification from network devices
> - **Log Aggregation** - Centralizing logs from multiple sources
> - **Alert Tuning** - Optimizing alert sensitivity and reducing false positives
