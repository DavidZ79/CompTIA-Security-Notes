## Log Data Collection

Essential log sources for security investigations:

<details>
<summary><strong>System and Application Logs</strong></summary>

*Primary log sources for investigation support*

**Log Types:**
- **Firewall Logs** - Network traffic filtering records
- **Application Logs** - Application-specific event records
- **Endpoint Logs** - Individual device activity records
- **Windows Logs** - Windows system event logs
- **OS Security Logs** - Operating system security events
- **IPS/IDS Logs** - Intrusion detection and prevention records
- **Network Logs** - Network infrastructure activity
- **DNS Logs** - Domain name resolution records
- **Web Server Logs** - Web application access records

</details>

<details>
<summary><strong>Log Metadata</strong></summary>

*Additional context information for log analysis*

**Metadata Elements:**
- **Timestamps** - When events occurred
- **Source Information** - Origin of log entries
- **User Context** - Associated user accounts
- **System Context** - Related system information

</details>

## Network Analysis

Network-level evidence collection and analysis:

<details>
<summary><strong>Packet Capture Analysis</strong></summary>

*Network traffic analysis for investigations*

**Packet Capture Uses:**
- **Traffic Analysis** - Analyzing network communications
- **Protocol Analysis** - Examining network protocols
- **Timeline Reconstruction** - Building chronological event sequences
- **Evidence Preservation** - Maintaining network evidence integrity

</details>
